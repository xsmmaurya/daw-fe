# DAW Frontend (Next.js)

## Features
- Rider panel: request rides, view live status
- Driver panel: go online/offline, accept/start/complete rides
- WebSocket client
- Zustand auth store
- Axios API wrapper

## Run
```
npm install
npm run dev
```

## Env
```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080/api/v1
NEXT_PUBLIC_WS_URL=ws://localhost:8080/api/v1/ws
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
